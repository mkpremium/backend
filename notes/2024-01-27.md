1. Escalar api & events worker a 0
2. Apagar Couchbase
3. Encender instancia de Couchbase con backups (couchbase-backups-4)
4. Crear indice para primario y `_documentType` en el bucket a migrar:
   - 'CREATE PRIMARY INDEX ON backup_actual'
   - 'CREATE INDEX idx_documentType_backup_actual ON backup_actual(_documentType)'
4. Trigger de backup de Couchbase deployando events-worker con los siguientes parámetros:
   - Command: `node,scripts/manual-interventions/publish-command-to-save-documents-in-postgres.js`
   - Env. vars:
       - COUCHBASE_BUCKET=backup_actual
       - COUCHBASE_URI=couchbase://10.0.31.203:8091?n1ql_timeout=20
       - COUCHBASE_USER=couchbase
       - DOCUMENTS_CONDITION=_documentType NOT IN ["system-preferences","history", "operator-refresh_token","operator-stats"]
       - FROM_COUCHBASE=backup_actual
       - ADD_ONLY=true
   - Trigger event worker to start copying documents from couchbase to postgres
     - Env vars:
       - COUCHBASE_BUCKET=backup_actual
       - COUCHBASE_URI=couchbase://10.0.31.203:8091?n1ql_timeout=20
       - COUCHBASE_USER=couchbase
